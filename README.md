### Haskcasting
A typechecked Haskell DSL for Hexcasting.

Pattern definitions (modules in `Haskcasting.Patterns.*`) were AI-generated and hand verified using data from the [pattern registry](https://book.hexxy.media/v/2.7.4/registry.json) (except for Hexical, which was scraped from the [docs](https://hexical.hexxy.media/v/1.5.0/1.0.0/en_us)).
